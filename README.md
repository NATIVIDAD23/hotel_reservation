# 🏨 Hotel Reservation System

This project is a **Hotel Booking Reservation Website** that helps manage hotel bookings, room inventories, and guest information.  
The platform is designed to simplify the reservation process, providing guests with a convenient way to secure their accommodations while allowing hotel staff to efficiently manage operations.

---

## 🚀 Features

- ✅ User Authentication (Guests, Admins, Staff)
- ✅ Multi-branch Hotel Management
- ✅ Room Management (pricing, availability, capacity, status)
- ✅ Booking & Reservation Workflow (pending → confirmed → check-in → check-out)
- ✅ Payment Tracking (cash, card, gcash, etc.)
- ✅ Guest Information Management & Booking History
- ✅ Reports & Analytics (revenue, occupancy rate, guest stats)
- ✅ Responsive UI with modern design

## 🛠️ Technologies Used

### **Backend**
- [Laravel 12](https://laravel.com/) – PHP framework for backend & APIs
- [Laravel Breeze](https://laravel.com/docs/12.x/starter-kits#laravel-breeze) – Authentication scaffolding
- [Laravel Sanctum](https://laravel.com/docs/12.x/sanctum) – API authentication
- [MySQL](https://www.mysql.com/) – Relational database

### **Frontend**
- [React](https://react.dev/) – Component-based UI
- [Vite](https://vitejs.dev/) – Fast build tool
- [Tailwind CSS](https://tailwindcss.com/) – Utility-first CSS framework
- [ShadCN UI](https://ui.shadcn.com/) – Pre-built, customizable UI components
- [Lucide React](https://lucide.dev/) – Icon library

### **Other Tools**
- [Composer](https://getcomposer.org/) – PHP dependency manager
- [Node.js & NPM](https://nodejs.org/) – JavaScript runtime & package manager
- [Git](https://git-scm.com/) – Version control
